## Halfeld zsh theme

> Minimalist zsh theme

![screenshot](/screen.png)


### How to install

```
curl https://raw.githubusercontent.com/IgorHalfeld/halfeld-zsh-theme/master/halfeld.zsh-theme --output $HOME/.oh-my-zsh/themes/halfeld.zsh-theme
```

__~/.zshrc__
```
ZSH_THEME="halfeld"
```
